# Welcome to Redstring

**Redstring** is a sophisticated React-based cognitive interface for constructing and navigating networks of interconnected **Things**. It implements a web-based system where Things can contain other Webs, enabling hierarchical recursion and composition for knowledge representation and exploration.

## Architecture Overview

Redstring uses modern web technologies to deliver a powerful cognitive interface:

- **React 18** with functional components and hooks
- **Zustand** for performant state management with Maps
- **React DnD** for drag-and-drop interactions
- **Framer Motion** for smooth animations
- **Vite** for fast development and building
- **Web Workers** for canvas operations
